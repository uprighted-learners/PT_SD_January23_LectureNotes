Intro to Git & GitHub
Command Description 
git init
 Initializes our repo(repository) on our machines 
git status
 Displays tracked files and their status 
git add
or 
git add .
 Stages any changed files, add to the commit 
git commit -m "Your message"
 Clears the staging and provides the "why", a label for what the commit contains 
git push
 Uploads your local commits 
git pull
 Downloads or "pulls down" any changes from GitHub 
git remote add origin https://github.com/userName/nameOfGitHubRepo
 Connects URL origin to local project 
git branch -M main
 Renames the master branch to main 
git push -u origin main
 For initial push, sends code up repo for the first time.

    When Updating a Repo

  • git add .
  • git commit -m "message here"
  • git push origin main
